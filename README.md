# Laravel starter

This is a inventory  project. You can download the zip from here [Laravel](https://github.com/Mostak1/acrh-inventory) or 
```bash
git clone https://github.com/Mostak1/acrh-inventory.git
```
run this command.

Install the Composer
```bash
git composer install
```
Next you need to make a copy of the `.env.example` file and rename it to `.env` inside your project root.

Run the following command to generate your app key:
```bash
php artisan key:generate
```
Coinnect The Database with Project from .env

For creating table run 
```bash
php artisan migrate
```
Then start your server:

```bash
php artisan serve
```

Your Laravel starter project is now up and running!
